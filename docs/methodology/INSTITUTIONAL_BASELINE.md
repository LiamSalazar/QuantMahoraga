# Institutional Baseline Methodology

La baseline oficial congela una arquitectura long-only ya promovida:

- `BASE_ALPHA_V2`
- `PARTICIPATION_ALLOCATOR_V2`
- `CONVICTION_AMPLIFIER_LAYER`
- `LEADER_PARTICIPATION_LAYER`
- `RISK_BACKOFF_LAYER_V2`
- continuation como filtro de calidad

La promoción no surgió de una tesis nueva, sino de una fase de acceptance y una promotion gate disciplinada sobre `14.3R`.
